Spatial Memory (Tversky, 1992)         j1/j2/j3 navigation works immediately
- Humans remember relative positions    - Users intuitively understand numbered jumps
- Not absolute coordinates              - No training needed for fish-eye navigation
- Mental maps use landmarks             - Notebook names become cognitive landmarks

Temporal Memory (Tulving, 1972)        Cross-time search feels natural
- Episodic vs semantic memory           - Activity view (episodic) vs search (semantic)
- Memory exists across time dimensions - Searching deleted/historical items obvious
- Time isn't linear in cognition       - "Find that thing I deleted last month" works

Procedural Memory (Squire, 1992)       Commands become reflexes
- Skills stored separately from facts   - j3 becomes automatic, not conscious
- Repetition encodes to basal ganglia   - C,V,B,S migrate to muscle memory
- No conscious recall needed            - Users forget they're "using an app"

Progressive Disclosure (Norman, 1988)  Features discover themselves
- Information should reveal as needed   - C, V, B first, then S, J, T naturally
- Avoid overwhelming users              - No feature overwhelm, natural progression
- Complexity layers                     - Novice to expert path without instruction

Chunking Theory (Miller, 1956)         Path segmentation matches cognitive limits
- Working memory: 7±2 items             - Display shows optimal segments
- Chunking aids recall                  - .../segment1/segment2/... pattern natural
- Memory consolidation requires chunks  - Git commits chunk history naturally

Dual Coding Theory (Paivio, 1971)      Numbers + words reinforce each other
- Verbal + visual channels separate     - [1] (verbal) + "note title" (verbal+visual)
- Dual encoding strengthens memory      - Location and name remembered together
- Retrieval via either channel          - Find by number OR by search

Cognitive Load Theory (Sweller, 1988)   Zero extraneous UI
- Intrinsic, extraneous, germane load   - Only intrinsic load remains
- Extraneous load impedes learning       - No menus, no toolbars, no distractions
- Germane load enables schema formation - Patterns emerge from usage

Recognition Over Recall (Craik, 1972)   Numbers trigger recognition
- Recognition faster than recall         - [1] instantly recognizable
- Less cognitive effort                  - No need to remember "where is that note?"
- Familiarity cues                       - Repeated exposure reinforces

Gestalt Principles (Wertheimer, 1923)   Interface organizes itself
- Proximity: related items grouped       - Notes together, subs together
- Similarity: consistent formatting      - All items look alike
- Closure: missing info inferred         - .../ indicates deeper path
- Figure-ground: foreground items clear  - Content stands out from UI

Cognitive Dissonance (Festinger, 1957)  No feature bloat
- Inconsistency causes discomfort        - App has no conflicting features
- Users seek coherence                   - Everything works the same way
- No mental conflict                     - One mental model suffices

Flow Theory (Csikszentmihlyi, 1990)    Writing without interruption
- Challenge-skill balance                - Commands match intentions
- Clear goals                            - Each keystroke has predictable result
- Immediate feedback                      - UI updates instantly
- Concentration                           - No distractions break focus
- Loss of self-consciousness              - App disappears
- Altered time perception                 - Hours feel like minutes

Attention Restoration (Kaplan, 1995)    Terminal provides "soft fascination"
- Directed attention fatigues             - No flashing, moving elements
- Involuntary attention rests              - Terminal recedes to background
- Restorative environments                 - Writing space feels calm

Prospect-Refuge Theory (Appleton, 1975) Terminal as cognitive shelter
- Prospect: overview of environment       - See all notebooks at home
- Refuge: safe, enclosed space            - Terminal as protected writing space
- Balance enables exploration              - Jump to explore, back to write
